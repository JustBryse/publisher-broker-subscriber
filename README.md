    * cd build
    * cmake ..
    * make
* Run Instructions:
    * cd build
    * make ./src/pub_bro_sub
* Testing Instructions:
    * cd build
    * make test
* Testing Dependecies:
    * GoogleTest (GTest)
        * libgtest-dev
        * libgmock-dev
